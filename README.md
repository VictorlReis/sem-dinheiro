# Descrição do projeto

Sem Dinheiro é um projeto de código aberto de controle de finanças pessoais e organização financeira. Atualmente existe um deploy feito utilizando a Vercel que você pode conferir clicando [aqui](https://sem-dinheiro.vercel.app/).

## Tecnologias utilizadas:

- [Next.js](https://nextjs.org)
- [NextAuth.js](https://next-auth.js.org)
- [Prisma](https://prisma.io)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)
- [Shadcnui](https://ui.shadcn.com/)
- [openAI](https://openai.com)
